Coding assignment: Exploring a simple Python shell

Assignment brief

Create a CLI/ shell that implements the following:
- When you enter the command LIST it lists the contents of the current directory
- The ADD command will add the following two numbers together and provide the result
- The HELP command provides a list of commands available
- The EXIT command exits the shell

Submission


import os

def main():
	while True:
		command = input("$ ")
		if command == "EXIT":
			break
		elif command == "LIST":
			print(os.listdir())
		elif command == "ADD":
			num1 = int(input("Enter first number: "))
			num2 = int(input("Enter second number: "))
			num3 = num1 + num2
			print(str(num1) + " + " + str(num2) + " = " + str(num3))
		elif command == "HELP":
			print("Enter LIST to list all files in the current directory.")
			print("Enter ADD to add two numbers together.")
			print("Enter HELP to list the available commands.")
			print("Enter EXIT to exit the application")
																	
if '__main__' == __name__:
main()

What are the two main security vulnerabilities with your shell?
- There is no input validation
- The application can list the contents of the current directory, regardless of permissions.
