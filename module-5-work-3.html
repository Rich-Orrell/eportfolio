Data Structures Reflection

Assignment brief

Think about online systems which I use on a daily basis and consider data structures used at the back-end

Submission

Data structures for online systems I use on a daily basis:

- Online file storage

o Queues used for uploading new content

- Online chess

o Arrays used to store game history

o Linked lists used to store game moves sequentially

- Search engines

o Trees for DNS searches

o Stacks for storing page history for navigating to previously accessed pages

- Online banking

o Hash tables for sensitive data

o Trees for storing transaction history
